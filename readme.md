# Heart Disease Predictor using Logistic Regression  

This is a **Streamlit-based web application** that predicts the risk of heart disease based on patient health parameters using **Logistic Regression**. Users can input health information such as age, blood pressure, cholesterol, and more, and the app predicts whether the patient is at risk of heart disease.  

---

## Features  

- Interactive **web UI** built with **Streamlit**.  
- Predicts **Heart Disease** based on key health parameters.  
- Handles **missing values** and **categorical features**.  
- Displays **prediction result** clearly (Risk / No Risk).  
- Can be extended to **multiple diseases** in future.
